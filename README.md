# Trello_coding
clone Trello

## Trello 클론 코딩

#### 서버 : nodejs + express

#### 프론트 : JavaScript

```bash
npm install express --save
```

```bash
node index.js
```

express generator 설치

```bash
npm install express-generator -g
```

## 프로그램 실행

```bash
nodemon app.js
```
